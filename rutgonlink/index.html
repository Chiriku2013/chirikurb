Rút Gọn Link — Chiriku Roblox

Nhập tên (ví dụ: Tên link), một ô thêm ("gì đó cũng đc, tùy") và link cần rút gọn. Nhấn Create để tạo link rút gọn. (Bản demo dùng API rút gọn công khai — link rút gọn sẽ có thể truy cập được bởi mọi người.)

Tên sẽ được lưu kèm (nếu bạn chạy backend riêng). Ở bản demo này tên chỉ dùng để hiển thị.

Lưu ý:

  • Trang này sử dụng shrtco.de public API để rút gọn link (demo). Link ngắn tạo ra bởi API đó là global — mọi người có thể truy cập URL rút gọn.
  • Để lưu thông tin (name / extra) cho tất cả mọi người (không lưu local), bạn cần một backend (API) hoặc dịch vụ lưu trữ công khai (ví dụ: Firebase, a small Node.js server với DB). Ở cuối file có hướng dẫn mẫu để bạn triển khai backend nếu muốn.

Nếu bạn muốn, tôi có thể tạo luôn hướng dẫn để deploy backend (Node.js + tiny JSON DB hoặc Firebase). Bạn sẽ cần một server để "lưu cho tất cả mọi người".
